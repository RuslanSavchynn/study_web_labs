# Character & Car — JS OOP 

Лабораторна робота для знань з об'єктно-орієнтованого програмування (ООП) у JavaScript, включаючи:
- класи та спадкування
- функції-конструктори з прототипами
- методи зміни контексту: ".bind()", ".call()", ".apply()"

# Структура

Увесь код знаходиться в одному файлі: "index.js".

Лабораторну роботу поділено на три частини:

# 1. "GameCharacter" (класи ES6+)
- Основний клас: "GameCharacter"
- Спадкоємці: "Warrior", "Mage"
- Методи: "get/set" для властивостей, "move()", "attack()", "castSpell()"

# 2. "Car" (функція-конструктор + прототипи)
- Основний конструктор: "Car"
- Спадкоємці: "ElectricCar", "SportsCar"
- Методи: "get/set", "move()", "charge()", "boost()"

# 3. Демонстрація:
- ".bind()" — зберігання контексту для відкладеного виклику
- `.call()" — виклик функції з явним контекстом
- ".apply()" — аналог `call()`, але з масивом аргументів

---

# Як запустити в терміналі

node index.js
