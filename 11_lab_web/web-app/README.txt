# GitHub Users Web App

Веб-додаток для роботи з API GitHub, включаючи:
- відображення списку користувачів
- пошук за логіном
- адаптивний дизайн у стилі GitHub

# Структура

Увесь код знаходиться в директорії `web-app`.

Проєкт поділено на три частини:

# 1. "index.html"
- Основна структура сторінки
- Поле пошуку
- Контейнер для списку користувачів

# 2. "style.css"
- Стилізація карток користувачів
- Адаптивні колонки (Grid)
- Спінер для завантаження

# 3. "main.js"
- Запити до API (`fetch`)
- Відображення даних і пошук
- Обробка помилок



# Як запустити

Через термінал використовуючи команду:

npx serve .

- Локально за адресою: http://localhost:3000

- Лережею за адресою: http://your_ip_adress:3000
(для доступу з інших пристроїв у локальній мережі)
